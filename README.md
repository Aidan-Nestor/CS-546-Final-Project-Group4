# Neighborhood Watch NYC

A web application for viewing and discussing NYC 311 service requests by ZIP code. Built for CS-546 Final Project.

## Features

- Search NYC 311 incidents by ZIP code with filters
- View trending incidents based on engagement
- Interactive maps showing incident locations
- Admin interface for data ingestion from NYC 311 API

## Installation

1. Install dependencies:
```bash
npm install
```

2. Run:
```bash
npm start
```

Visit `http://localhost:3000`

## Usage

- Enter a ZIP code to search incidents
- Click incidents to view details and comments
- Log in to add comments and vote
- View trends at `/trends`
- Admin data ingestion at `/admin/ingest`
